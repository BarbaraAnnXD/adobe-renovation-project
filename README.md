# House Renovation Project

## Goal

Renovate and organize the house so it is safe, functional, affordable, and ready to move into.

## Main Priorities

1. Water/plumbing
2. Electrical and AC
3. Walls and holes
4. Flooring
5. Cleaning and sealing
6. Final move-in setup

## Current Status

The house has had major repairs completed, including roof work, siding, bug sealing, framing burned areas, doors, and floor planning. Current focus is finishing utilities, patching walls, flooring gaps, AC wiring, and making the house livable.
